# 📊 NumPy Student Performance Analyzer

A beginner-friendly data analysis project built entirely using NumPy to analyze student performance across multiple subjects.

## 🔍 What it does
- Calculates average marks per student and per subject
- Identifies topper and weakest student
- Finds most consistent performer using standard deviation
- Filters students above 75% average using boolean masking
- Identifies elite students using 90th percentile

## 🧠 NumPy Concepts Covered
- `np.array()`, `.shape`, `.dtype`
- `np.mean()`, `np.sum()`, `np.std()` with axis=0 and axis=1
- `np.argmax()`, `np.argmin()`
- Boolean Masking for filtering
- `np.percentile()` for ranking analysis

## 🛠️ Tech Stack
Python | NumPy | Google Colab

## 🚀 How to Run
Open `student_analysis.py` in Google Colab or any Python environment and run directly.
